## 步骤 2：分批导入 kangxi_dict（24,097 条）

从第 8001 条开始，每批 1000 条：

```powershell

# 第 2 批：9001-10000

# ... 继续直到 24097
```

## 步骤 5：导入 classics_entries（124,120 条）

这个表有数组类型字段，需要特殊处理：

```powershell
# 第 1 批：1-1000
psql -h localhost -U postgres -d seekname_db -c "COPY (SELECT id, book_id, book_name, chapter_name, ancient_text, modern_text, array_to_string(keywords, '|') as keywords FROM classics_entries ORDER BY id LIMIT 1000 OFFSET 0) TO STDOUT WITH CSV HEADER" | psql $VERCEL_URL -c "COPY classics_entries(id, book_id, book_name, chapter_name, ancient_text, modern_text, keywords) FROM STDIN WITH CSV HEADER"

# ... 继续，共约 125 批
```

## 简化方案

如果分批太麻烦，可以直接用 pg_dump 导出后导入：

```powershell
# 1. 导出单个表
$env:PGPASSWORD='postgres'
pg_dump -h localhost -U postgres -d seekname_db --data-only --inserts -t kangxi_dict > kangxi_data.sql

# 2. 导入到 Neon（可能需要多次尝试）
$env:PGPASSWORD='npg_2WiMHoA4RdTQ'
psql $VERCEL_URL -f kangxi_data.sql
```
