```
/personal              → 个人起名表单 → /naming?surname=...      (AI真实评分)
/company/form          → 公司起名表单 → /company?name=...         (易经数理)
/brand/form            → 品牌起名表单 → /brand?name=...          (调性标签)
/shop/form             → 店铺起名表单 → /shop?name=...           (温馨亲切)
/pet/form              → 宠物起名表单 → /pet?name=...            (宠物类型)
/evaluate/form         → 名字测评表单 → /evaluate?name=...      (四维评分)
```
