## 🐛 构建修复

### 核心问题：pg 原生驱动错误
**根因**：`evaluate/page.tsx` 是 "use client"，却动态 import 了 `@/lib/prisma`。
pg 驱动依赖 Node.js 内置模块（fs/dns/net/tls），浏览器里不存在。

**修复方案**：
1. `src/app/evaluate/page.tsx` — 移除 prisma 导入，改为调用新的 API Route
2. `src/app/api/name/evaluate/route.ts` — 新建服务端测评 API（含 DB 降级 fallback）
3. `src/lib/prisma.ts` — 顶部加 `import "server-only"`，防止 client 代码误导入
4. `evaluate/page.tsx` — 添加服务降级提示（离线时纯 JS 评分）
