## 新增 API 路由
- `src/app/api/blog/posts/[id]/route.ts` — 单篇文章查询（含阅读量 +1）
- `src/app/api/blog/comments/route.ts` — GET/POST/DELETE 评论（含树形结构回复）
- `src/app/api/blog/like/route.ts` — GET/POST 点赞（toggle 逻辑）

- BlogComment 模型：支持子评论（parent_id 自引用）、likeCount 计数
- BlogLike 模型：userId + targetType + targetId 唯一约束
- BlogFavorite 模型：userId + postId 唯一约束
- User 模型添加 comments/likes/favorites 反向关联

## 迁移 SQL
- `prisma/migrations/202604160900_add_blog_interaction_tables/`：评论/点赞/收藏三表 + 触发器 + seed 用户

- 支持增量运行（slug 已存在则跳过）
- 自动创建/关联 BlogTag 和 BlogPostTag

- `src/app/api/upload/image/route.ts`
