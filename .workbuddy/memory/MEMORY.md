# WorkBuddy 长期记忆

## 项目：寻名（AI 智能起名平台）

### 项目概述
- 类型：网站（PC/H5，Next.js）+ 微信小程序
- 核心功能：AI 智能起名（人名/品牌名/宠物名等）
- 技术栈：Next.js + PostgreSQL + 大语言模型 API + 微信支付 + 支付宝

| 代号 | 职责 |
|------|------|
| 寻名顾问 | 常驻学术顾问，精通汉字姓名磁场学、通书历法、历史典籍，提供规则与建议由工程师实现 |
