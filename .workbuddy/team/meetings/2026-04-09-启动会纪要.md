## 一、项目规模确认

**汇报人**：寻名PLM

### 1.1 平台覆盖
- **PC Web端**：桌面浏览器访问，注重SEO与信息展示
- **H5移动端**：手机浏览器适配，与小程序形成互补
- **微信小程序**：微信生态内原生体验，便于分享传播

### 1.2 核心功能模块
| 模块 | 功能描述 |
|------|---------|
| 智能起名 | 基于AI生成候选名字，支持人名/品牌名/宠物名 |
| 文化筛选 | 结合磁场学、周易、五行进行名字评分与筛选 |
| 支付系统 | 微信支付 + 支付宝，支持单次购买/会员订阅 |
| 用户中心 | 起名记录、收藏、个人资料管理 |
| 分享功能 | 生成精美分享卡片，支持微信好友/朋友圈 |

### 1.3 项目定位
- **性质**：MVP（最小可行产品）起步，验证市场后迭代扩展
- **团队**：8人协作（含顾问），扁平化管理

## 五、AI技术融入起名过程

**汇报人**：寻名AI工程师、寻名顾问

### 5.1 AI起名流程设计

```
用户输入需求
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│  第1层：信息收集                                            │
│  - 起名类型（人名/品牌名/宠物名）                            │
│  - 姓氏（如适用）                                           │
│  - 生辰八字（可选，用于五行分析）                             │
│  - 风格偏好（古典/现代/文艺/大气等）                         │
│  - 特殊要求（避讳字、必含字等）                              │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│  第2层：AI生成（多模型策略）                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  Prompt结构（分层设计）                              │   │
│  │  ├── 系统层：顾问提供的文化规则（磁场学/周易/五行）    │   │
│  │  ├── 场景层：起名类型特定的生成策略                   │   │
│  │  └── 用户层：个性化参数注入                          │   │
│  └─────────────────────────────────────────────────────┘   │
│  主模型：GPT-4o（创意与质量）                                │
│  备用模型：DeepSeek（成本与速度）                            │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│  第3层：文化筛选（顾问规则代码化）                           │
│  - 磁场评分：按字形/音韵/笔画计算磁场强度                    │
│  - 周易卦象：匹配卦象并生成解读                              │
│  - 五行补缺：根据八字分析推荐补益五行的字                    │
│  - 典籍溯源：验证名字出处，过滤负面典故                      │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│  第4层：结果呈现                                            │
│  - 候选名字列表（带评分排序）                                │
│  - 每个名字的详细解读（寓意/磁场/卦象/出处）                  │
│  - 收藏/分享/购买操作入口                                    │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 顾问文化规则体系

**寻名顾问**提供的专业维度：

| 维度 | 规则说明 | AI应用方式 |
|------|---------|-----------|
| **汉字姓名磁场学** | 分析字形结构、笔画数理、音韵五行的能量场 | 生成后按磁场强度评分排序，低分淘汰 |
| **周易卦象** | 六十四卦与名字格局对应，卦爻辞解读人生寓意 | 为每个候选名匹配最佳卦象，AI生成解释文案 |
| **四柱八字** | 根据出生年月日时推算五行缺失与喜用神 | AI结合八字结果，优先推荐补益五行的字 |
| **通书历法** | 农历干支、纳音五行、生肖宜忌、节气影响 | 作为辅助参考，影响名字的时令适配性 |
| **历史文化典籍** | 《诗经》《楚辞》《说文解字》《周易》《尔雅》《礼记》 | AI从典籍中提取典故，为名字赋予文化出处 |

### 5.3 协作机制

```
寻名顾问（文化专家）
    │
    ├── 输出《磁场评分规则文档》──────┐
    ├── 输出《周易卦象匹配规则》──────┤
    ├── 输出《五行补益策略》──────────┤──→ 寻名AI工程师（代码实现）
    ├── 输出《典籍字词库》────────────┤
    └── 输出《筛选算法伪代码》────────┘
                │
                ▼
    ┌───────────────────────────────┐
    │  转化为：                      │
    │  - Prompt系统层指令            │
    │  - 后端筛选算法                │

    └───────────────────────────────┘
```
