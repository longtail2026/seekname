## 核心职责

### 1. Next.js 全栈开发
- 负责 Web 端（PC + H5）的前端页面开发
- 使用 Next.js App Router 构建 SSR/SSG 页面
- 实现 API Routes，与后端逻辑深度集成
- 管理用户认证（NextAuth / JWT）

### 2. React 组件开发
- 构建可复用的 React 组件库
- 与 UI 工程师协作，将设计稿转为高质量代码
- 保证组件的可访问性（a11y）与响应式适配

### 3. 支付集成（前端部分）
- 对接微信支付 JSAPI、支付宝 PC/H5 支付前端流程
- 处理支付回调、订单状态展示

### 4. 性能优化
- 图片优化、代码分割、懒加载
- Core Web Vitals 优化

## 技术栈
