# 寻名网首页设计规范

## 一、设计概述

### 1.1 设计理念
- **主题**：健康活力 + 古典文化融合
- **核心视觉**：橙色主调传达活力，回纹元素体现文化底蕴
- **目标**：打造专业、可信赖的起名服务平台形象

### 1.2 参考基准
- 基于现有参考网站 `C:\seekname - 副本\seekname-website` 的结构
- 调整视觉风格以匹配新的品牌要求

```

```

#### 下拉菜单结构
**个人起名**：
- 宝宝起名
- 成人改名
- 起名（通用）

**公司起名**：
- 公司起名
- 品牌起名
- 项目起名

#### 样式规范
- 高度：72px
- 背景：白色 + 底部阴影
- Logo区域：左侧，234×60 banner
- 导航项：14px，黑色，悬停变橙色
- 下拉菜单：白色背景，圆角8px，阴影

### 4.2 回纹分割线

#### CSS实现
```css
.pattern-divider {
  height: 20px;
  background-image: url("data:image/svg+xml,..."); /* 回纹SVG图案 */
  background-repeat: repeat-x;
  opacity: 0.3;
}
```

#### 使用场景
- 各区块之间的分隔
- 重要内容的上下装饰

## 五、首页区块结构

### 5.1 Hero Banner 区
- **高度**：视口高度的 70-80%
- **背景**：渐变（橙到暖白）或背景图
- **内容**：
  - 大标题：寻一个好名，许一个未来
  - 副标题：AI智能起名，融合传统文化与现代科技
  - CTA按钮：「免费起名体验」（橙色主按钮）

### 5.2 服务入口区
- **布局**：4列网格（桌面）/ 2列（平板）/ 1列（手机）
- **卡片内容**：
  1. 个人起名（宝宝/成人）
  2. 公司起名（公司/品牌/项目）
  3. 宠物起名
  4. 名字测评

### 5.3 AI特色展示区
- **背景**：品牌橙色
- **内容**：七步AI起名流程图解

### 5.4 文化典籍区
- **装饰**：回纹分割线
- **内容**：展示文化典籍数据库优势

### 5.5 用户评价区
- **布局**：3列卡片
- **内容**：用户 testimonial

### 5.6 Footer
- **背景**：深灰/黑色
- **内容**：网站信息、联系方式、版权

## 八、交互规范

### 8.1 按钮状态
| 状态 | 样式 |
|------|------|
| 默认 | 橙色背景，白色文字，圆角8px |

- 下拉菜单：鼠标悬停 200ms 延迟展开
- 移动端：点击汉堡菜单展开全屏导航

### 8.3 页面滚动
- 平滑滚动（smooth scroll）
- 导航栏固定顶部（sticky header）

## 九、开发注意事项

1. **性能优化**：图片使用 Next.js Image 组件，启用懒加载
2. **SEO**：每个区块使用语义化 HTML 标签
3. **无障碍**：按钮和链接有明确的 focus 状态
4. **兼容性**：支持 Chrome、Firefox、Safari、Edge 最新两个版本
