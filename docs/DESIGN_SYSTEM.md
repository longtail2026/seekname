# 寻名网 UI 设计系统
## 「墨韵流金」Moyun Liujin Design System

---

## 一、设计理念

### 1.1 核心定位

**愿景**：打造一个「见字如面、名以载道」的起名平台——现代极简的交互体验中，蕴含着东方美学的深厚底蕴。

**关键词**：
- **现代极简** — 去除繁杂装饰，追求简洁高效的交互
- **东方气韵** — 水墨意境、金色点缀、留白美学
- **专业信赖** — 科学严谨的分析背后，是对传统文化的尊重

### 1.2 设计哲学

| 维度 | 现代 | 传统 | 融合方式 |
|------|------|------|----------|
| **布局** | 栅格系统、响应式 | 中轴对称、留白 | 响应式栅格 + 东方居中美学 |
| **色彩** | 低饱和度、渐变色 | 朱砂、墨色、金色 | 莫兰迪色为底，朱砂金色点缀 |
| **字体** | 无衬线正文 | 宋体标题 | 思源黑体正文 + 思源宋体标题 |
| **图形** | 几何图形、图标 | 回纹、云纹、印章 | 现代线条图标 + 传统边框装饰 |
| **动效** | 微交互、弹性动画 | 水墨晕染、落笔 | ease-out 曲线 + 墨韵渐入动画 |

### 1.3 品牌精神

> **「寻一个好名，许一个未来」**

三个核心意象：
- 🔮 **墨** — 知识底蕴，文化根基（品牌色：墨色 #1A1A2E）
- ✨ **金** — 吉祥美好，品质追求（品牌色：流金 #D4AF37）
- 🌊 **水** — 灵动智慧，生命流动（品牌色：青瓷 #4A7C7E）
