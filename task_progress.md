# 英文起名发音匹配优化任务

## 问题分析
对于"徐小燕"→"Xu Xiao Yan"，当前引擎的问题：
1. **多音节匹配缺失**：引擎只匹配了"yan→ian"在"Diana"中就得高分，完全忽略了"xiao"音节没有匹配
2. **无全音节覆盖要求**：2音节名字只匹配1个音节就能得0.9分，这不对
3. **语义搜索干扰**：语义搜索找到大量不相关的常见英文名，发音匹配分数高的反而排在后面
